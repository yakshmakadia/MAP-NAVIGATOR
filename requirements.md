## Installation & Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/map-navigator.git
   cd map-navigator
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the program:
   ```bash
   python map_navigator.py
   ```

## requirements.txt
```
networkx
matplotlib
heapq
```
